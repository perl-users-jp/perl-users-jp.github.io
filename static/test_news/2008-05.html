perl-mongers.org がはじまりました

perl-mongers.org

みんなで Perl を楽しく使えたらいいね！という目的のもと、perl-mongers.org をはじめます。

OpenID でログインして気軽に Perl の事を書ける趣旨だそうです。
blog でやると過去の良記事が埋もれやすいので perl-users.jp 側で上手く保管できればいいなと思ってます。

せっかくの org ドメインだから日本にこだわらなくてもいいんじゃないか? と思ってますがどうでしょう :)

Moose が XS なしでうごくようになりました

http://search.cpan.org/~stevan/Moose-0.45/

Moose 0.45 がリリースされたのですが、このバージョンから XS 不要に
なりました。Pure Perl ですべて動くようになりましたので、XS がつか
えないような環境でも安心して使うことができます。

今迄 XS なモジュールのインストールが面倒そうだなあとおもって敬遠し
ていた方もこれを期に Moooooooooooooooooooooooose してみませんか?

CodeRepos のアカウントをもってる Perl 使いの方ならば気軽にこのニュースに記事を投稿できます。

TPF-J構想

lestrratさんが素晴らしい構想を立ち上げたので紹介です。

http://mt.endeworks.jp/d-6/2008/05/tpfj.html

TPF-JはThe Perl Foundation Japanとして、Perlの普及活動および、普及活動を行う任意の個人・団体に人的、資金的な援助を行う目的で活動する。

協力者が必要そうなので、興味ある方はコンタクト取ってみて下さい。

たとえば先日立ち上がったperl-users.jpにしても（彼らが実際に賛同してくれるかどうかは別として）

lestrratさんも、その中の人だと思ってる。

autobox 2.50 がリリースされました

http://search.cpan.org/~chocolate/autobox-2.50/

autobox 2.50 がリリースされました。

このバージョンから INTEGER, FLOAT, NUMBER, STRING という
パッケージがサポートされるようになりました。

今迄はスカラー変数は SCALAR というパッケージにひもづけられていましたが、
これからは整数、浮動小数点数、数値、文字列それぞれにたいしてメソッドを
定義できるようになりました。
