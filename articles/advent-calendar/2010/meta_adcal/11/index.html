クリスマスの六本木は人大杉だとおもったtokuhiromですこんにちは。

正直、最近はてな記法つかってなくて、まじ忘れてるし、はてな記法だけしかサポートしてないのってイケてなくね?っておもったので、markdown で書けるようにした。 また、mattn さんによって、meta data のうめこみができるようになっているから、

    タイトル
    meta-format: markdown
    meta-author: tokuhirom

    markdown markdown

みたいなかんじでかけばいいようにした。

markdown を html に変換するには Text::Markdown をつかえばいい。

    use Text::Markdown qw/markdown/;

    $html = markdown($text);

ってやるだけ!かんたんでしょ。
