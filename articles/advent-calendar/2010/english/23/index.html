Test::SharedFork - testing with fork(2)

tag

Test::SharedFork is utility module for Test::Builder. This module makes forking test! This module merges test count with parent process & child process.

The following code is works well!

    use Test::More tests => 200;
    use Test::SharedFork;

    my $pid = fork();
    if ($pid == 0) {
        # child
        ok 1, "child $_" for 1..100;
    } elsif ($pid) {
        # parent
        ok 1, "parent $_" for 1..100;
        waitpid($pid, 0);
    } else {
        die $!;
    }
